How to make a 404 page (& other things to do with .htaccess)

.htaccess is a cool format

.htaccess is one of the main formats used in web development, so it's very important that you know how to use it. It basically just uses plaintext, but there are a few rules.
  1. The rules in the file apply to the current and sub directories.
  2. The file can only be name '.htaccess'. ONLY '.htaccess'. Nothing else.
